Sencha Touch - HTML5 Mobile App Framework

Welcome to Sencha Touch

Sencha Touch is a JavaScript framework for creating applications that look and feel native on Google Android and Apple iOS devices. It features beautiful user interface components and rich data management, all powered by the latest HTML5 and CSS3 web standards. Deploy web-based or wrap for distribution on mobile app stores.
